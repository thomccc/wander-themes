# Fonts

This folder is for **local font files** that you want to serve via Vite.

## Vend Sans (used by `/style-three`)

`/style-three` is configured to use a font-family named **Vend Sans** for both title and body.

If you have Vend Sans font files, add them here using these filenames (recommended):

- `VendSans-Regular.woff2` (400)
- `VendSans-Medium.woff2` (500)
- `VendSans-SemiBold.woff2` (600)
- `VendSans-Bold.woff2` (700)

Optional fallbacks (if you donâ€™t have `.woff2`):

- `VendSans-Regular.woff`
- `VendSans-Medium.woff`
- `VendSans-SemiBold.woff`
- `VendSans-Bold.woff`

Once added, refresh `http://localhost:3000/style-three` and it should render using Vend Sans.
